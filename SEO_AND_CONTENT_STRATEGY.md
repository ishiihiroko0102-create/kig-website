## 1. SEO 戦略の決定

### 1.1 背景・課題分析

**現状：**
- Google で「株式会社KIG」検索時に 4 ページ目以降に表示
- 1-3 ページ目：警備会社のKIG（kig.jp）と自動車部品買取の競合企業（パワーグループ）が占領

**分析結果：**
- ユーザーの企業（kig-ltd.com）：設立 2010年、資本金 8,000万円、大阪・九州を中心に全国対応
- 競合企業（パワーグループ）：大企業、複数事業展開、規模感演出
- **「株式会社KIG」単独での 1 位獲得は現実的ではない**（同名企業との競合、ドメイン差）

---

### 1.2 新 SEO 戦略（確定）

#### **目標キーワード（複合キーワード戦略）**

**Phase 1（3ヶ月後）**
```
✓ 「KIG 買取」：1-5 位
✓ 「自動車パーツ 買取 大阪」：5-10 位
✓ Google マップで「大阪 買取」：上位表示
```

**Phase 2（6ヶ月後）**
```
✓ 「KIG 買取」：1 位（確定）
✓ 「自動車パーツ 買取」：1-3 位
✓ 「大阪 買取」「九州 買取」：各地域 1 位
✓ 「大阪 出張買取」「九州 パーツ買取」等：地域複合キーワード上位
✓ 「BMW 買取」「中古パーツ 買取」等：5-10 位以内
✓ 埼玉営業所（検討中）：展開時に「埼玉 買取」も同様に施策予定
```

#### **戦略の根拠**

1. **ユーザーの企業の強み**
   - ドメイン古い（2010年 vs 競合 2016年+）
   - 資本金大きい（8,000万円）
   - 営業所が複数かつ全国対応（大阪・九州を中心）
   - ドメインはすでに一定の信頼度がある

2. **競合との差別化ポイント**
   - 「自動車部品買取」という専門分野では、ユーザーが専門
   - 「KIG 買取」などのニッチキーワードでは、ユーザーが確実に 1 位可能
   - 警備会社とは業界が異なるため、複合キーワードで競合しない

3. **ビジネス上の有効性**
   - 実際の顧客は「株式会社KIG」で検索しない
   - 「買取」「パーツ」「大阪」など実務的なキーワードで検索
   - ユーザーが「KIG 買取」で 1 位なら、顧客はちゃんと見つけられる

## 3. 企業の強み・ユニークさ（ユーザー入力）

### 3.1 ユーザーの KIG vs 競合企業との差別化

**ユーザーの企業：**
- 出張買取が強み（営業マンが直接買取に行く）
- 郵送買取も実施（見積もり気軽、送料当社負担）
- 営業人の質（知識豊富、礼儀正しい、素早い）
- WEB事業部があることで社内システムが完全に完結
- 複数事業展開のポテンシャル（将来アパレル事業など）

**競合企業（パワーグループ）：**
- 大企業スケール感を演出
- 複数事業を並列表示
- 企業理念「人と地球の未来のために。リサイクルで、笑顔をつなぐ。」
- **買取方法の詳細説明がない** ← 弱点
- **個人的な対応感がない** ← 弱点

### 3.2 ユーザー企業の独自性

1. **郵送買取システム**
   - 見積もりが気軽
   - 送料は当社負担
   - 全国どこからでも対応可能

2. **営業人の質による差別化**
   - 知識の豊富さ
   - 礼儀正しさ
   - 素早い対応
   → 「人」を強みにする（企業スケール感ではなく）

3. **WEB事業部による完結性**
   - 社内にシステム構築・運用部門がある
   - セキュリティ・信頼性が高い
   - 他社よりも対応が速い

4. **複数事業への進化**
   - 現在：自動車パーツ買取
   - 将来：アパレル事業への対応可能
   - ビジネスの多角性と安定性をアピール

## 4. ターゲット別メッセージ（ユーザー入力）

### 4.1 BtoB営業先向け

**メインメッセージ：**
```
「呼ばれたらすぐに行きます。どこまでも。」
```

**伝えたいこと：**
- レスポンスの速さ
- 信頼性・ビジネスパートナーとしての確実性
- 営業マンの質（知識・礼儀）
- 継続的なサポート体制

### 4.2 BtoC買取希望者向け

**メインメッセージ：**
```
「細やかなアフターサービス。業界最安値を目指す。」
```

**伝えたいこと：**
- 透明性のある査定
- 丁寧な対応
- 価格の公正性
- 郵送買取なら手軽に利用可能
- 送料無料の負担（顧客メリット）

### 4.3 求職者向け

**メインメッセージ：**
```
「AI とテクノロジーを武器に、人のチカラを活かす。」
```

**伝えたいこと：**
- WEB事業部があることで、テクノロジー企業としての一面
- 営業人のキャラクター・やりがい
- セカンドチャンス制度などの人事施策
- 複数事業への成長機会
- 新しい価値観の企業文化

## 5. 競合分析（パワーグループ）

### 5.1 競合企業の特徴

**企業名:** 株式会社パワーグループ
**ホームページ:** https://www.powergroup.jp/renew/

**戦略的特徴：**
1. 企業理念を前面に押し出す（「人と地球の未来のために。リサイクルで、笑顔をつなぐ。」）
2. 複数事業を並列表示（スケール感の演出）
3. 企業規模を強調（本社東京、複数事業所）
4. モダンなデザイン（国際的、彩度低下フィルター等）

**弱点：**
1. **買取方法の詳細説明がない** - トップページには記載されていない
2. **郵送買取などの利便性の説明がない** - ユーザーのメリット訴求が弱い
3. **営業人の個人的な対応感がない** - 大企業風で、親密感が低い
4. **詳細情報がサブページに分散** - ユーザーが情報を探す手間がある

### 5.2 ユーザーの差別化ポイント

| 要素 | パワーグループ | ユーザーのKIG |
|------|----------|---------|
| **企業理念** | 環境・社会貢献を前面 | 顧客サービス・営業人の質 |
| **スケール感** | 大企業風（規模を強調） | 営業人の質（個人対応） |
| **買取方法** | 詳細説明なし | 出張買取 + 郵送買取（詳細説明） |
| **営業対応** | 企業としての対応 | 営業マンの個人的な対応 |
| **デジタル対応** | デザイン重視 | WEB事業部による完結性 |

## 6. コンテンツ作成方針

### 6.1 各ページのテーマ（次フェーズで詳細化予定）

| ページ | テーマ | 狙うキーワード |
|--------|-------|----------|
| **トップ** | 営業マンの質 + 買取方法の多様性 | KIG, 買取, パーツ |
| **About** | 「どこまでも行きます」+ WEB事業部 | KIG 企業, 信頼 |
| **Business** | 出張買取 + 郵送買取の具体説明 | 買取方法, 自動車パーツ |
| **Recruit** | 営業人のやりがい + テクノロジー | 採用, 買取営業 |
| **営業所ページ** | 各地域での買取実績 | 大阪買取, 九州買取, 大阪出張買取 |
| **ニュース** | 買取事例・YouTube・Tips・キャンペーン | パーツ名 + 買取, BMW買取等 |

### 6.2 ニュース記事の考え方

**月 4～8 件の短編ニュース投稿**

**運用方針:**
- 50～150 語程度の短編ニュース形式
- 無理のない継続可能な頻度を優先
- 「毎月ブログ更新が止まる」より「定期的に少量投稿」が信用につながる

**投稿テンプレート例（新入荷情報）：**
```
【新入荷】BMW E90 ヘッドライト入荷しました！

先日、大阪営業所で買取した BMW E90 のヘッドライトが入荷しました。
状態も良く、すぐに販売できる商品です。

BMW パーツをお探しの方、お気軽にお問い合わせください。
https://kig-ltd.com/contact

#BMW買取 #自動車パーツ #大阪買取
```

**投稿テンプレート例（YouTube 動画）：**
```
【動画】郵送買取の流れを 3 分で解説！

ご自宅から一歩も出ずに自動車パーツを売却できる「郵送買取」。
その流れを動画でわかりやすく解説しました。

初めての方も安心。送料はすべて当社負担です。
https://youtube.com/kig-channel

#郵送買取 #自動車パーツ #買取方法
```

**各投稿に必ず含める要素：**
- 狙うキーワード（3～4 個）
- 営業所名または地域（SEO 効果 + ローカル検索対応）
- CTA リンク（問い合わせ / 動画 / 詳細ページへ）
- ハッシュタグ（SNS 連携で拡散効果）
